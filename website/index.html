Scientific Software

Scientific software refers to the algorithms, simulations, statistical analyses, data processing and data visualization operations that are used to accomplish scientific products.

Page maintained by Daniel Garijo

An increasing amount of scientific work relies on computational experiments, including domains ranging from Pharmacology to Geosciences, Astronomy or Computer Science.
